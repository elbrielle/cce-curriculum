# CCE Facilitator Guides -- Development Notes

## Overview

This is a draft of 36 weekly Facilitator Guides covering all 6 six-week periods (Week 0 through Capstone). Each one includes TEKS alignment, teacher scripting, DOK questions, warm-ups, exit tickets, assessments, and ELL supports. Format is based on the Vet Tech lesson template.

Source materials: Hats & Ladders (public website + limited student login), eDynamic Learning (public course page only), Xello, BLS/CareerOneStop, the Irving ISD CTE Pathways poster, and the original pacing spreadsheet.

This is a starting foundation, not a finished product. Use what works, edit what does not, toss what misses.
